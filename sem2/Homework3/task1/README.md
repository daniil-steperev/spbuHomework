# Домашняя работа 3, задание 1 #
Задача: 
> Реализовать класс для работы с хеш-таблицей (на списках).
 Общение с пользователем должно происходит в интерактивном режиме:
 добавить значение в хеш-таблицу, удалить значение из хеш-таблицы,
 поиск значения в хеш-таблице, показать статистику по хеш-таблице
 (общее число ячеек, load factor, число конфликтов, максимальная
 длина списка в конфликтных ячейках и т.п.), заполнить хеш-таблицу
 содержимым файла, выбрать хеш-функцию для подсчета хеша (из заранее
 заданных в коде). Смена хэш-функции должна происходить во время работы
 программы, в класс используемая хеш-функция должна передаваться из клиентского кода.

Используемый интерфейс:
* HashFunction

Используемые классы:
* HashTable
* List
* PolynomialHashFunction
* SummaryHashFunction
* ElementAbsenseException
* SimilarHashFunctionException
* WrongInputException

> А также тесты к классу HashTable.

### Решение: ###
1. Создаём интерфейс HashFunction.
2. Реализуем HashFunction в классах SummaryHashFunction и PolynomialHashFunction.
   * SummaryHashFunction: считает сумму кодов элементов и выводит ее по модулю mod.
   * PolynomialHashFunction: считает хеш в виде: a1*x^n + a2*x^(n-1) + ... по модулю mod, где aI - код элемента, x - power.
3. Создаем классы исключений:
   * WrongInputException (пользователь пытается ввести некорректные данные)
   * ElementAbsenseException (попытка удалить отсутсвующий элемент)
   * SimilarHashFunctionException (попытка сменить текущую хеш-функция на такую же)
4. Создаем класс HashTable
   * метод changeHashFunctions работает по следующему принципу:
     + происходит сравнение переданной хеш-функции и текущей; если они совпадают, то выбрасывается исключение SimilarHashFunctionException
     + если хеш-функции разные, то переопределяем таблицу элементов, высчитывая хеш по новой функции
   * метод getStatistics возвращает:
     + количество элементов в хеш-таблице
     + load-factor
     + количество конфликтов (когда по одному хэш-коду определяется больше одного элемента)
     + максимальная длина конфликтной ячейки
     + количество пустых ячеек в хэш-таблице
5. Пишем main и тесты к хеш-таблице.
