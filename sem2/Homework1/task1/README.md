# Домашняя работа 1, задание 1 #
Задача: 
> Создать класс, реализующий функциональность стека.

Используемые классы:
* Stack
** Node

### Решение: ###
Создаём класс Stack, внутри которого находится private класс Node (узел, элемент стека). Реализуем основные метода стека:
* push()
* pop()
* print()
А также основные методы в классе Node, которые пригодятся для работы со стеком:
* два конструктора
* getValue()
* setNextNode()
