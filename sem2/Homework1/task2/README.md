# Домашняя работа 1, задание 2 #
Задача: 
> Создать класс, реализующий функциональность односвязного линейного списка.

Используемые классы:
* List
* Node

### Решение: ###
Создаём класс List, внутри которого находится private класс Node (узел, элемент списка). Реализуем основные методы списка:
* два метода add() (добавить на позицию, добавить в конец)
* два метода remove() (убрать по индексу, убрать последний)
* find() (найти заданный элемент; если он есть - вернуть его индекс, иначе вернуть -1)
* printList()


А также основные методы в классе Node, которые пригодятся для работы со списком:
* два конструктора
* getValue()
* setNextNode()
