# Домашняя работа 4, задание 2 #
Задача: 
> Реализовать AVL дерево, реализующее интерфейс Collection.

Используемые классы:
* AVLTree
* TreeIterator
* Node
* NodeOperator

> А также тесты к классу AVLTree.

### Решение: ###
1. Создаем классы Node (узел, элемент дерева).
2. Создаем класс NodeOperator. Этот класс предназначен для работы с узлами дерева:
   * добавление нового элемента (add); если элемент меньше текущего, то добавим к его левому потомку, если больше - к правому
   * удаление выбранного элемента (remove); удаление происходит по следующему принципу:
     + если у удаляемого элемента нет потомков, то зануляем (делаем ссылку null) его;
     + если у удаляемого элемента есть только один потомок, то заменяем элемент на него;
     + если у удаляемого элемента есть оба потомка, то находим самый правый элемент левого потомка (rightest), удаляем его; меняем значение удаляемого на rightest
   * поиск нужного элемента (меньшие элементы лежат в левом поддереве, большие - в правом)
   * балансировка
     + левый и правый повороты
   * перевод узла в String
3. Создаем класс TreeIterator на основе ArrayList.
   > Добавляем все значения в список. Когда нужно получить новый элемент - берем первый из списка, проверка на hasNext() эквивалентна проверки ArrayList на isEmpty.
4. Создаем класс AVLTree. Реализем методы addAll, removeAll, containsAll, toArray, retainAll. 
5. Пишем тесты.
