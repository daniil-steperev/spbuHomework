# Домашняя работа 4, задание 1 #
Задача: 
> Переделать один из реализованных в прошлой домашней работе список
 в виде generic'а. На основе него реализовать класс UniqueList,
 который не содержит повторяющихся значений. Реализовать классы
 исключений, которые генерируются при попытке добавления в такой
 список уже существующего или при попытке удаления несуществующего элемента.

Используемые классы:
* List
* UniqueList
* AbsenceElementException
* AlreadyAddedElementException
* EmptyListException
* WrongIndexException

> А также тесты к классу List, UniqueList.

### Решение: ###
1. Создаем классы-исключения: 
   * AbsenceElementException (отсутствие в списке элемента, над которым необходимо осуществить некоторые операции)
   * AlreadyAddedElementException (попытка добавить уже присутствующий в списке элемент (актуально для класса UniqueList))
   * EmptyListException (попытка осуществить операции над пустым списком)
   * WrongInputException (введен неправильный индекс, по которому должна быть осуществлена операция над списком)
2. Создаём класс List.
3. На основе класса List создаем класс UniqueList (односвязный список, в котором нет повторяющихся элементов)
4. Пишем тесты к классу List, UniqueList.
