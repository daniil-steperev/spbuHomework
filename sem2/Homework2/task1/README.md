# Домашняя работа 2, задание 1 #
Задача: 
> Создать интерфейс сортировщика, принимающего массив элементов 
и выполняющего над ним сортировку. На его основе создать несколько классов, 
реализующих произвольные (разные) алгоритмы сортировки.

Используемый интерфейс:
* Sorter (сортировщик)

Используемые классы:
* BubbleSort (сортировка пузырьком)
* SelectionSort (сортировка выбором)
* ShakerSort (коктейльная сортировка)

> А также тесты к каждому классу.

### Решение: ###
1. Создаём интерфейс сортировщика.
2. Создаём классы сортировок, реализующие сортировщика.
3. Пишем тесты к каждому классу:
  * тест на сортировку пустого массива
  * тест на сортировку массива из одного элемента
  * тест на сортировку массива, состоящего из одинаковых цифр
  * тест на сортировку отсортированного по возрастанию/убыванию массива
  * тест на сортировку обычного массива
