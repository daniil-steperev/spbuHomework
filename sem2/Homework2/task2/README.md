# Домашняя работа 2, задание 2 #
Задача: 
> Разработать интерфейс, представляющий структуру данных "стек".
 На его основе реализовать стек двумя разными способами.
 С его помощью реализовать стековый калькулятор для
 подсчета арифметических выражений.

Используемый интерфейс:
* Stack

Используемые классы:
* ArrayStack (стек на основе массива)
* ListStack (стек на основе списка)
* StackCalculator (калькулятор для выражений в постфиксном виде на основе стека)
* WrongInputException (исключение, которое должно вызываться, когда пользователь вводит некорректные данные)

> А также тесты к каждому классу (кроме WrongInputException).

### Решение: ###
1. Создаём интерфейс стека.
2. Создаём класс ArrayStack, ListStack, реализующие интерфейс Stack.
3. Создаём класс StackCalculator на основе ListStack
  * метод calculateAnswer принимает выражение в постфиксной записи и считает результат по следующему принципу:
    - встретилось число - добавь в стек
    - встретилась операция - вытащи два последних элемента из стека, выполни операцию, добавь в стек
  > В случае некорректной работы (например, нужно достать элемент из стека, а его там нет) вызывается исключение WrondInputException
4. Пишем тесты.
