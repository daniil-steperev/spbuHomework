# Домашняя работа 2, задание 2 #
Задача: 
> Разработать интерфейс, представляющий структуру данных "стек".
 На его основе реализовать стек двумя разными способами.
 С его помощью реализовать стековый калькулятор для
 подсчета арифметических выражений.

Используемый интерфейс:
* Stack

Используемые классы:
* ArrayStack (стек на основе массива)
* ListStack (стек на основе списка)
* StackCalculator (калькулятор для выражений в постфиксном виде на основе стека)
* Converter (переводит строку из инфиксной формы в постфиксную)
* EmptyStackException (исключение, которые должно вызываться, когда используется метод pop() из пустого стека)
* WrongInputException (исключение, которое должно вызываться, когда пользователь вводит некорректные данные)

> А также тесты к каждому классу (кроме EmptyStackException, WrongInputException).

### Решение: ###
1. Создаём интерфейс стека.
2. Создаём классы для исключений: EmptyStackException, WrongInputException.
3. Создаём класс ArrayStack, ListStack, реализующие интерфейс Stack.
4. Создаём класс Converter
   * метод convertToPostfix осуществляет перевод выражения из инфиксной формы в постфиксную
     - сперва используем метод setSpaces, чтобы *правильно* расставить пробелы: после каждой операции и операнда ставим пробел (это понадобится при разделении выражения на операнды и операции)
     - перевод выражения осуществляем по следующему принципу:
       + встретилось число - добавим в строку postfixString
       + если встретилась операция - добавляем в стек
       + если встретилась закрывающаяся скобка - добавляем все из стека в postfixString, пока не встретим открывающуюся скобку (convertFromBrackets)
       + в конце добавляем всё из стека в postfixString (convertFromStack)
5. Создаём класс StackCalculator на основе ListStack
   * метод calculateAnswer принимает выражение в инфиксной записи, с помощью метода convertToPostifx переводит выражение в постфиксную форму и считает результат по следующему принципу:
     - встретилось число - добавь в стек
     - встретилась операция - вытащи два последних элемента из стека, выполни операцию, добавь в стек
  > В случае некорректной работы (например, нужно достать элемент из стека, а его там нет) вызывается исключение EmptyStackException, WrongInputException (например, после работы калькулятора в стеке осталось больше одного элемента)
6. Пишем тесты.
