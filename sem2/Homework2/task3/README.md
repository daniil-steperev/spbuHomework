# Домашняя работа 2, задание 3 #
Задача: 
> Дан массив размерностью N x N, N - нечетное число.
 Вывести элементы массива при обходе его по спирали, начиная с центра.
 Для решения задачи разработать интерфейс Выводилка с методом вывести(),
 реализовать на его основе два класса, осуществляющих либо вывод на
 консоль либо в файл. Написать программу, которая по желанию пользователя
 выбирает реализацию Выводилки и выводит массив.

Используемый интерфейс:
* SpiralWriter (выводилка)

Используемые классы:
* ConsoleWriter, реализующий SpiralWriter (вывод в консоль)
* FileWriter, реализующий SpiralWriter (вывод в файл)
* Matrix (класс матрицы)
* MatrixUnpacker (класс, который *раскрывает* матрицу по спирали)
* Main
* WrongInputException (исключение, которое должно вызываться, когда пользователь вводит некорректные данные)

> А также тесты к классу Matrix, MatrixUnpacker.

### Решение: ###
1. Создаём интерфейс выводилки.
2. Создаём класс WrongInputException (будем вызывать это исключение, когда возникнет ситуация некорректного ввода)
3. Создём класс Matrix
  * конструктор класса запрашивает у пользователя размер матрицы и саму матрицу
    - если размер матрицы четный - выбрасывается исключение WrongInputException (противоречие условию задачи)
4. Создаём класс MatrixUnpacker
  * метод convertMatrixToList переводит матрицу из табличного вида в массив, раскрывая матрицу с середины по спирали по следующему принципу:
    - пока размер сдвига меньше, чем размер строки/столбца делаем фазу:
      + фаза состоит из движения в одну из четырёх сторон на величину сдвига (shift)
    > Фазы выполняются в таком порядке: вправо, вниз, влево, вверх, вправо, ...
5. Создаём класс ConsoleWriter, реализующий интерфейс SpiralWriter.
6. Создаём класс FileWriter, реализующий интерфейс SpiralWriter.
7. Создаём класс Main, взаимодействующий с пользователем (0 - вывод *раскрытой* матрицы в консоль, 1 - в файл).
8. Пишем тесты.
